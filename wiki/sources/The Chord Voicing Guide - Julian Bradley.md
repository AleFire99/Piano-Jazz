# The Chord Voicing Guide — Julian Bradley

**Format:** PDF ebook (11 pages)
**Author/Teacher:** Julian Bradley (Jazz Tutorial)
**Core focus:** Five types of jazz piano chord voicings, how to choose between them, and how to practice them

---

## Key takeaways

- Every jazz chord can be voiced many ways; 5 professional-level types are covered
- Three voicing **principles** apply to all types: no repeated notes, root/3rd/7th at the bottom, leave space for the melody
- Voicings are demonstrated on the three main 7th chord types (maj7, min7, V7) — all others are adaptations
- **Melody-Matching** is Julian's technique for choosing which voicing to use: pick the voicing whose top note matches the current melody note
- Practice one voicing type at a time via **side-slipping** (transpose around all 12 keys before moving on)
- Count **interval patterns**, not note names, when memorizing voicings
