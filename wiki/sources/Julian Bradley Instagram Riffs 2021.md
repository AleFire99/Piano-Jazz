# Julian Bradley Instagram Riffs 2021

**Format:** PDF (collection of short riff lessons from social media)
**Author/Teacher:** Julian Bradley
**Core focus:** Short self-contained riffs demonstrating bebop scales, modal interchange, bossa nova bass lines, ripple effect voicings, and 5th harmonization

## Key takeaways

- **Bebop scale**: adds one chromatic passing note to a standard scale so that chord tones land on strong beats. Example — G dominant bebop: G A B C D E F F♯ (8 notes; the added F♯ ensures G, B, D, F all fall on downbeats).
- **Bossa nova bass line**: left hand plays root + 5th of each chord in the characteristic syncopated bossa rhythm. Foundation for playing Latin jazz.
- **Modal interchange**: temporarily borrowing chords from a parallel mode (e.g., using chords from C minor while in C major). Creates surprising color changes without modulating.
- **Ripple effect**: arpeggiate a chord voicing in a rolling wave upward — gives energy to static harmonies and is idiomatic in ballad and bossa contexts.
- **Melody doubled in 5ths**: harmonize a melody by adding a note a perfect 5th above each melody note. Creates a full, resonant texture.
