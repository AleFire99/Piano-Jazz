# The Chord Symbol Guide — Julian Bradley

**Format:** PDF reference sheet (1 page)
**Author/Teacher:** Julian Bradley (Jazz Tutorial)
**Core focus:** Complete reference for jazz chord symbol notation used in lead sheets

---

## Key takeaways

- Six 7th chord types matter in jazz: maj7, min7, V7, minor-major 7, half-diminished, diminished 7
- 6th chords: 6th *replaces* the 7th — Cm6 uses a **major 6th** (A natural), not a minor 6th (Ab)
- All extensions (9, 11, 13) are built from the major scale; altered extensions (b9, #9, #11, b13) appear only on V7 chords
- Notation convention: use `#4` for maj7 chords, `11` for min7/V7 (not interchangeable)
- Sus4 *replaces* the 3rd; 11th chord *adds* 4th alongside the 3rd — different sounds
- C7b13 implies the full **C altered** scale (C Db Eb E Gb Ab Bb)
- Slash chords: chord/bass note — right hand plays chord, left hand plays bass note
- C7(#9) = "The James Bond Chord"
